# Leave

A short platformer made for **Hack Club Campfire Organiser Training** with the theme: **Stuck in a Loop**.

## How to Play

- **Move:** Arrow Keys or WASD
- **Jump:** W or Up Arrow
- **Goal:** Find the exit

## About

You're stuck on a platform. There are spikes. There are signs. And there's a door somewhere.

Keep going.
